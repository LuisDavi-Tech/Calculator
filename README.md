# Calculator
## Uma calculadora simples
---
<p>Para clonar o repositório basta rodar o comando abaixo em seu terminal</p>

```sh

git clone https://github.com/LuisDavi-Tech/Calculator/

```
---

<Br>

```sh
sudo apt install git
```
